# Hyoban's config

This is the configuration repository I need to start a React project, which includes the following:

1. ESLint configuration
2. Prettier configuration
3. tsconfig basic configuration
4. Vite configuration
5. Tailwind preset
6. Collection package of utility and hooks functions
7. Component library package using shadcn ui

Use following command to create a new Vite + React project:

```bash
npm create vite@latest
```

And then follow the next steps to setup all the configurations you need.
